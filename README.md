# Biker Zone Landing Page

This is the **Landing Page** of the Biker Zone website built using **Tailwind CSS** and **DaisyUI**.  
It includes a fully responsive header, mobile-friendly dropdown menu, hero section with carousel slider, and footer.

## Features
- ✅ Responsive Header & Navbar
- ✅ Hero / Carousel Section
- ✅ Call-to-Action Button
- ✅ Footer
- ✅ Mobile-first Design

## Technologies Used
- HTML5
- Tailwind CSS
- DaisyUI
- JavaScript (optional, for carousel navigation)
